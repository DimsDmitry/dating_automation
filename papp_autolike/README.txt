1) Устанавливаете python! Инструкция по установке будет ОТДЕЛЬНО

2) Запускаете Python

3) В командной строке по очереди вбиваете команды:
pip install pyautogui
pip install opencv-python

4) Открываете веб-версию сайта pure, настраиваете все параметры поиска (на ПК версии может потребоваться VPN,
индивидуально)

5) ОТМАТЫВАЕТЕ СТОЛБЕЦ СЛЕВА ВНИЗ, чтобы в поле монитора не попало сердечко и надпись "у тебя пока нет лайков"
Иначе это сердечко будет сбивать работу скрипта

6) Запускаете файл pure_modified_v1.py, оставляете экран включенным
Скрипт умеет обрабатывать и закрывать всплывающие окна, но баги и вылеты всё равно могут быть
Сообщайте о них в ЛС.

P.S. Первые 3 пункта следует выполнить только один раз!
P.S.S. Проверьте чтобы масштаб страницы браузера был 100%
