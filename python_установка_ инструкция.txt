1) Заходим на сайт https://www.python.org/downloads/

2) Downloads -> Python 3.12.6 (подойдёт любая другая актуальная версия)

3) Скачиваем установщик, запускаем

4) Ставим галочку "Add Python to PATH"

5) Жмём "Costumize installation"

6) В следующем окне проверяем наличие галочек везде, жмём Next

7) В следующем окне проверяем, чтобы путь установки Python был максимально коротким, желательно прямо в диск C
Например:

C:\Python310

Этот пункт необязателен но повысит эффективность

8) Жмём Next, устанавливаем. Теперь можно запускать файлы формата .py
